# Starter Vue 2 / Vuex

It's an update of my VueJs 1.X.X stack for working with Vue 2.
Example of product List / Card from Vuex

### Installation

You must have gulp
```sh
npm i gulp -g
```

To work on the project, clone the repo and

```sh
$ yarn install (or npm install)
$ gulp
```

For production

```sh
gulp build
```
